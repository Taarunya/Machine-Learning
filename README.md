# 🧠 Machine Learning Models Collection

This repository contains implementations of multiple **Machine Learning algorithms** including regression, classification, clustering, and deep learning models.
The project demonstrates how different algorithms perform on real-world datasets such as **Kidney Disease** and **Housing Data**.

---

## 📂 Project Structure

